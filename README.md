# LinkPeek
LinkPeek is a browser-based contextual extension that helps identify suspicious URLs, phishing patterns, dangerous downloads, redirects, typo-squatting, homoglyph spoofing, and deceptive web behavior before interaction. 

## Installation
Clone the repository
Open chrome://extensions
Enable Developer Mode
Click Load Unpacked
Select the extension folder
Disclaimer

LinkPeek performs passive structural and contextual analysis. It is not an antivirus or guaranteed phishing blocker.
